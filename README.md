# Cadastro de Produtos

Projeto desenvolvido em Python com o objetivo de praticar conceitos fundamentais da linguagem.

## Funcionalidades

* Cadastrar produtos
* Listar produtos cadastrados
* Buscar produtos por nome ou ID
* Editar informações de produtos
* Remover produtos
* Calcular o valor total em estoque

## Conceitos praticados

* Listas
* Dicionários
* Funções
* Loops (`while` e `for`)
* Condicionais (`if`, `elif`, `else`)
* Tratamento de exceções (`try` e `except`)
* Organização de código
